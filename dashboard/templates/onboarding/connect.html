{% if platform == "facebook" %}

Connect your Facebook Page

You will need a Page Access Token from Meta's Graph API Explorer. Follow these steps:

1
Open Graph API Explorer

Go to developers.facebook.com/tools/explorer

2
Select your app

Pick your app from the dropdown (or create one at developers.facebook.com/apps)

3
Get Page Access Token

Switch "User or Page" to your Page, then click "Generate Access Token"

4
Add permissions

Add pages_manage_posts, pages_read_engagement, and pages_read_user_content

5
Copy the token and your Page ID

Find your Page ID in Page Settings → About → Page transparency

Back
{% elif platform == "instagram" %}

Connect your Instagram Business Account

You need an Instagram Business or Creator account linked to a Facebook Page.

1
Link Instagram to a Facebook Page

In Instagram Settings → Account → Linked accounts → Facebook

2
Get a token from Graph API Explorer

Same process as Facebook. Add instagram_basic, instagram_content_publish

3
Find your Instagram User ID

Query /me/accounts then /{page-id}?fields=instagram_business_account

Back
